And you are done


What it does:

It has some more built-in tags with the name of colors which may help you a lot by changing the colors

Example - <red>Red colored text</red>

Output - Red colored text


Another example - <blue>Blue colored text</blue>

Output - Blue colored text

Made the work easy, right


Colors which are supported:

Aqua, Black, Blue, Brown, Chocolate, Cornsilk, Crimson, Cyan, Darkblue, Darkcyan, Darkgray, Darkgreen, Darkorange, Darkred, Deepink, Gold, Gray, Green, Lightblue, Lightcyan, Lightgray, Lightpink, Lightyellow, Lime, Magenta, Maroon, Mediumblue, Navy, Orange, Orangered, Pink, Purple, Red, Royalblue, Silver, Skyblue, Teal, Tomato, Violet, Wheat, White, Whitesmoke, Yellow, Yellowgreen


The <random> tag:

Example - <random>Random colored text</random>

Output - Random colored text
